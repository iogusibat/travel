### 文字の装飾

メッセージを入力するときに以下のように文字の前後に付け加えることで、投稿されたメッセージのフォントを装飾することができる。

|     |     |
| --- | --- |
| >テスト1 | 頭に｜がついてインデントする。主にメッセージの引用などで使う。 |
| `テスト2` | インライン引用。ボーダー囲み、赤文字、背景グレー。 |
| “`テスト3“` | ボーダー付きボックス |
| _テスト4_ | イタリック（斜め） |
| *テスト5* |  ボールド（太字） |
